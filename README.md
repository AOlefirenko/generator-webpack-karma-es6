> A Yeoman generator to author libraries in ES2015 using webpack and karma (phantomjs)

## Installation

First, install [Yeoman](http://yeoman.io) and generator-webpack-karma-es-6 using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g generator-webpack-karma-es6
```

Then generate your new project:

```bash
yo webpack-karma-es6
```
